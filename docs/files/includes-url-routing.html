URL Routing – SEO-Friendly Filter URLs

Registers WordPress rewrite rules for filter endpoint URLs: /{archive-base}/filter/{tax1-slug1}+{tax1-slug2}/{tax2-slug1}/

URL segments are positional: position N corresponds to the Nth enabled filter group (sorted by order). An underscore _ placeholder skips a position so that later groups can still be addressed without creating ambiguous URLs.
